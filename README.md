## Начало работы

### Установка

Предоставьте пошаговые инструкции по установке вашего веб-сервиса:

```bash
# Клонирование репозитория
git clone git@github.com:kotopesp/sos-kotopes.git

# Переход в директорию проекта
cd sos-kotopes
```
